Something about the compatability issue:

I am trying to keep this website work on any web browser. But all web codes written are just recommendations, not commands for web browser. Deverlopers can not do too many things to handle the visual effects, because web browsers can just ignore any recommendation. So, when I am wring web codes, I will make sure the codes will work well with Firefox®, and try to make them compatible with Internet Explorer®, though the visual effects on IE is not guaranteed.
