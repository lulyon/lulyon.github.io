Welcome to this website. Recently I've been asking friends about naming this website, which should be special and never used by anyone. And I got what I think the ideal name for it: "吃货的世界: CyhoWorld"(eater's world).

Of course, this website is not about eating. But any way, I like eating. More importantly, people who come here, reading the materials is like eaters hunting something to eat. Moreover, giving a non-eating website a name eater's world sounds even more special.

Why create this website?

This website was originally created when I was intended to grasp basic technique on web development. Then I came up with the idea that the website should be simple and clean. I decided not using any template written by other developer and all the content should be written from scratch. Gradually, more stuff were added to the site.

As a psychological explaination of the motivation, people are unprecedented self-centered in modern times when one considers every aspect of one's life important and should be known by others. Sadly, I am no exception of these crowds of egocentrism. Thus, for me the site is not only an experimental field, some place to hack, but also the byproduct of social psychology, something to feed my vanity.
